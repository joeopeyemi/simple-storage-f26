# Simple Storage Smart Contract

## 📖 Description
A Solidity smart contract for storing and retrieving a number.

## 🛠 Tech Stack
- Solidity
- Foundry
- Ethereum

## ✨ Features
- Store data on-chain
- Retrieve stored value

## ⚡ Installation
git clone https://github.com/your-username/simple-storage-f26.git
cd simple-storage-f26
forge install

## ▶️ Usage
forge build
forge test
